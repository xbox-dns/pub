# Диагностика подключения к Xbox DNS

Если у вас проблемы с подключением — выполните шаги ниже и пришлите результаты в поддержку.

---

## Что понадобится

Все команды выполняются в **PowerShell** (правой кнопкой по Пуску → «Терминал» или «Windows PowerShell»).

Также скачайте утилиту **WinMTR**: [https://sourceforge.net/projects/winmtr/](https://sourceforge.net/projects/winmtr/)

---

## Наши серверы

| Сервер | IP |
|---|---|
| Основной | `111.88.96.50` |
| Резервный | `111.88.96.51` |

Выполните диагностику для того сервера, с которым у вас проблема. Если не уверены — делайте для обоих.

---

## Шаг 1. Ping-тест (проверка потерь пакетов)

Вставьте в PowerShell:

```
ping 111.88.96.50 -n 100
```

Дождитесь завершения (пару минут). В конце будет строка вида:
```
Пакетов: отправлено = 100, получено = 99, потеряно = 1 (1% потерь)
```

**Скопируйте весь вывод** и сохраните в текстовый файл.

---

## Шаг 2. Трассировка маршрута (WinMTR)

1. Откройте WinMTR
2. В поле **Host** введите: `111.88.96.50`
3. Нажмите **Start**
4. Подождите, пока счётчик дойдёт до **100** отправленных пакетов
5. Нажмите **Stop**
6. Нажмите **Export TEXT** и сохраните файл

---

## Шаг 3. Проверка доступности по протоколам

Вставьте в PowerShell по очереди:

```
Test-NetConnection 111.88.96.50 -Port 443
```

```
Test-NetConnection 111.88.96.50 -Port 53
```

```
Resolve-DnsName xbox.com -Server 111.88.96.50
```

Скопируйте вывод каждой команды.

---

## Шаг 4 (опционально). Дамп трафика

Этот шаг нужен только если попросит поддержка.

Скачайте **Wireshark**: [https://www.wireshark.org/download.html](https://www.wireshark.org/download.html)

1. Откройте Wireshark
2. В поле фильтра захвата вверху введите: `host 111.88.96.50`
3. Выберите ваш сетевой адаптер и начните захват (синяя кнопка-плавник)
4. В другом окне PowerShell запустите: `ping 111.88.96.50 -n 50`
5. Подождите завершения пинга
6. Остановите захват в Wireshark (красный квадрат)
7. Сохраните файл: **Файл → Сохранить как** → формат `.pcapng`

---

## Что прислать в поддержку

1. ✅ Результат ping (текст из Шага 1)
2. ✅ Файл из WinMTR (Шаг 2)
3. ✅ Результаты Test-NetConnection и Resolve-DnsName (Шаг 3)
4. 📎 Файл .pcapng из Wireshark (Шаг 4, если просили)
5. ℹ️ Опишите проблему: что именно не работает, какая игра/сервис, с какого момента
